# Cuatro Vientos Nono 🌬️

Sitio web oficial de Cuatro Vientos Nono.

## 📋 Descripción

Este es un sitio web estático desarrollado para presentar información sobre Cuatro Vientos Nono.

## 🚀 Tecnologías

- HTML5
- CSS (en carpeta static)
- GitHub Pages para hosting

## 🌐 Dominio

El sitio está configurado con un dominio personalizado (ver archivo CNAME).

## 📁 Estructura del Proyecto

```
.
├── CNAME           # Configuración de dominio personalizado
├── favicon.png     # Ícono del sitio
├── index.html      # Página principal
└── static/         # Recursos estáticos (CSS, JS, imágenes)
```

## 🔧 Desarrollo Local

Para ejecutar el sitio localmente:

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. O usa un servidor local como `python -m http.server`
